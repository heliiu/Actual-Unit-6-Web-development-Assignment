MidShires Heritage Railway is a volunteer run organisation that maintains a number of steam locomotives that run over the “holiday season” on Less busy British Rail track.
This means there is no interference with normal services.

The rolling stock for Heritage railways includes a variety of options which include standard gauge quarry stock, locomotives, historic coaches and much more.

Locomotives

Locomotives provide the traction that pulls trains. Heritage railways usually operate a mix of steam locomotives, diesel locomotives, and sometimes diesel multiple units (DMUs)
